# StudioGL — Immersive 3D Web Experience

A high-performance, visually immersive 3D web experience built 
with React Three Fiber, GSAP, and custom GLSL shaders. StudioGL 
pushes the boundaries of what's possible in the browser — combining 
WebGL rendering, scroll-driven 3D animations, and cinematic page 
transitions for a premium creative experience.

## ✨ Features

- Custom GLSL shader materials for dynamic 3D surface effects
- Scroll-synchronized 3D scene transitions using GSAP ScrollTrigger
- Cinematic page transition animations via GSAP timeline orchestration
- High-fidelity 3D model rendering with React Three Fiber
- Smooth 60fps WebGL rendering with performance optimization
- Fully responsive 3D canvas layout

## 🛠️ Tech Stack

- **React** — component-based UI architecture
- **React Three Fiber (R3F)** — React renderer for Three.js WebGL
- **Three.js** — underlying 3D WebGL engine
- **GSAP + ScrollTrigger** — scroll-driven and timeline animations
- **GLSL** — custom vertex and fragment shaders
- **CSS3** — layout and typography
